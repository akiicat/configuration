
## Clone

```shell
git clone https://github.com/akiicat/vim.git
cd vim
```

## Install Vim

### Ubuntu

```shell
sudo add-apt-repository ppa:jonathonf/vim
sudo apt update
sudo apt install -y vim
```

### Mac

```shell
brew install vim
./install
```

```shell
cp ./bash_profile ~/.bash_profile
cp ./gitconfig ~/.gitconfig
```
