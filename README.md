
## Require

- git
- curl

```shell
sudo apt install -y git curl
```

## Install

### crul

```shell
curl -L http://get.akiicat.com | bash -s -- vim git tmux bash
```

### wget

```shell
wget -O - http://get.akiicat.com | bash -s -- vim git tmux bash
```
